# ZCode App - IDE Server

## Development

### Commands

install application
```
bin/node/install
```

Build application
```
bin/node/build
```

Client with Nodejs
```
bin/node/cli
```

Client with NestJs development
```
bin/node/nest
```

Client with NestJs debug
```
bin/node/nest_debug
```


NestJs unit tests
```
bin/node/test_unit
```

NestJs e2e tests
```
bin/node/test_e2e
```

Application logs
```
bin/node/logs
```
